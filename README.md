# Amna Shabir - Cybersecurity & AI/ML Portfolio

A modern, interactive portfolio showcasing cybersecurity, AI/ML, and networking engineering expertise. Built with a cyberpunk aesthetic featuring dark themes, monospace fonts, and smooth animations.

## 🚀 Features

### Interactive Project Showcase
- **4 Featured Projects** with detailed modals
- Click any project card to view comprehensive details
- **ReconKit** - Modular reconnaissance framework
- **PayloadForge** - Educational payload generation tool
- **IoT Attack & Defense Toolkit** - ESP32 firmware reverse engineering
- **Inventory System** - Full-stack Java/PHP application

### Project Details Include:
- 📋 **Description** - Comprehensive project overview
- 🛠️ **Tools & Technologies** - Complete tech stack
- 📝 **Build Instructions** - Step-by-step setup guide
- 📦 **Dependencies** - All required software/hardware

### Professional Sections
- **Hero Section** - Introduction with skill badges
- **About** - Professional background and education
- **Experience** - Internship timeline with achievements
- **Contact** - Direct links to email, GitHub, LinkedIn

### Design & UX
- 🌙 **Dark Cyber Theme** - Black background with green/cyan accents
- ⌨️ **Monospace Typography** - Tech-focused font styling
- ✨ **Smooth Animations** - Scanline effect and hover transitions
- 📱 **Responsive Design** - Mobile-optimized layout
- 🎯 **Interactive Modals** - Keyboard and click-to-close functionality

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in browser

### Installation

1. **Clone or Download**
   ```bash
   # Download the HTML file to your local machine
   # Or clone if using version control
   git clone [repository-url]
   cd portfolio
   ```

2. **Open in Browser**
   ```bash
   # Simply open the HTML file in your browser
   # Windows: Double-click amna_portfolio.html
   # Or drag file into browser window
   ```

3. **View Portfolio**
   - Navigate through sections using the top navigation
   - Click project cards to view detailed information
   - Use contact links to connect

## 📖 Usage

### Navigation
- **Top Navigation**: Jump to About, Projects, Experience, Contact sections
- **CTA Buttons**: "View Projects" and "Get in Touch" shortcuts
- **Smooth Scrolling**: Automatic scroll to sections

### Project Interaction
- **Click Project Cards**: Opens detailed modal with full information
- **Modal Controls**:
  - Click ✕ button to close
  - Click outside modal to close
  - Press Escape key to close

### Responsive Design
- **Desktop**: Full grid layouts and hover effects
- **Mobile**: Stacked layouts, touch-friendly interactions
- **Tablet**: Adaptive grid columns

### Content Updates
1. **Personal Info**: Update hero section and about content
2. **Projects**: Modify `projectDetails` object in JavaScript
3. **Experience**: Edit timeline items in HTML
4. **Contact**: Update links and information

### Adding New Projects
```javascript
// Add to projectDetails object
newProject: {
  title: 'Project Name',
  description: 'Project description...',
  tools: ['Tool1', 'Tool2', 'Tool3'],
  buildSteps: `Step-by-step instructions...`,
  dependencies: ['Dep1', 'Dep2', 'Dep3']
}
```
